# News Page - Challenge

## Setup

To run the project, just clone it and run:
```shell
yarn
```

After package installation, just run:
```shell
yarn start
```

## Tech

- React
- Typescript 4.9
- External Libs:
  - Axios
  - Auth0
  - Semantic UI React

## Improvements points

- LocalStorage Service better testing
